## Es6 Boilerplate

A boilerplate to get started with new es6 features using babel and gulp.

### Installation

    npm install
    gulp

You should be able to open the app at http://localhost:8001
    
### Getting Started

Add your code in src/js/app.js e.g

    let foo = "bar";

Your code will be compiled to 

    "use strict";
    
    var foo = "bar";

in build/js/app.js
